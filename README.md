### **This repository contains:**
- **Project 1:** Discriminative (Logistic Regression) and Generative model (GAN)
   - **Dataset:** MNIST dataset
- **Project 2:** Transformer Architecture (GPT-2, BERT, and T5 model)
   - **Dataset:** CNN Dailymail
- **Project 3:** Prompt Engineering Lab
   - **Model:** LLaMA 3.2 (1B) deployed locally via Ollama
   - **Task:** Summarization of a fixed paragraph
- **Project 4:** Comparative Training Methods (Pre-Training (unsupervised), Supervised Fine-Tuning (SFT), and Reinforcement Learning)
   - **Model:** Tiny GPT
   - **Dataset**: WikiText-2
- **Project 5: **Emerging Models Research

Models: Qwen2.5-VL-7B (multimodal vision-language model) and DeepSeek-Coder-6.7B (specialized code generation model).
Practical Demonstration: Environment
Platform: Google Colab (T4 GPU)
Framework: Hugging Face Transformers
Model: Qwen/Qwen2.5-VL-7B-Instruct (bfloat16 precision)
Test Cases

Case 1: Image-Based Question Answering
Input: Beach sunset image with 4 people jumping

Tasks:

Object counting and identification
Pose description and spatial reasoning
Scene context and emotion inference
Case 2: Chart/Graph Analysis
Input: Dual-axis bar/line chart (Units Sold vs. Total Transaction, Jan-Jun)
Tasks:

Dual-axis data extraction
Trend analysis and correlation detection
Structured JSON output generation
